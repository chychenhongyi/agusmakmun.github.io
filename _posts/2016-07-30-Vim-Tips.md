# 1. Motion[^1] #
*   简单的移动
    -   0 -> 到本行头
    -   ^ -> 到本行第一个不是空白字符的地方
    -   $ -> 到本行结尾
    -   g_ -> 到本行最后一个不是blank字符的地方
*   更有效率的移动
    -   NG -> 移动到第N行
    -   gg -> 移动到第一行 | 1G or :1
    -   G -> 到最后一行
    -   (w/e/W/E) -> 按照单词移动(开头/结尾/Blank分隔开头/Blank分隔结尾)
    -   %   ->  匹配括号移动，包括(,{,[
    -   */# ->  匹配光标所在的单词，移动到下一个/上一个匹配单词
    -   fa ->   到下一个为a的字符处，a可以换成其他字符，如fs
